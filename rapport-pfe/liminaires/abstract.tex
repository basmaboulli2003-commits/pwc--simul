\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This final-year project focuses on integrating the requirements of the PCI Secure Software Framework (PCI SSF) into a DevSecOps software development lifecycle. The objective is to propose an approach that improves application security, strengthens control traceability, and facilitates compliance assessment preparation.

The adopted methodology consists of analyzing the PCI SSF framework, studying the existing DevSecOps lifecycle, and mapping security requirements to development activities and expected compliance evidence. The project also emphasizes security control automation and continuous improvement of secure software development practices.

\textbf{Keywords:} PCI SSF, DevSecOps, software security, compliance, software lifecycle, HPS, PowerCARD.
